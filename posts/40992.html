Docker

简介

docker就好比传统货运集装箱, 统一了应用打包上云的方式

Docker 是管理容器的引擎

Docker 为应用、部署平台， 提供了虚拟化技术

VM VS Container

VM 相对笨重， guestOS 独立操作系统对CPU, 内存等资源消耗较大

共享Linux内核，使用 namespace 和 cgroup 来实现容器化的隔离和资源限制。

每种技术都有自己的运行场景

Docker优势

对于开发者而言 ： Build once, Run anywhere。

对运维人员而言 : Configure once、Run anything。
