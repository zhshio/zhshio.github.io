📚学长带你扫盲太理软件工程专业

Hey 大家好，我是 Shio👋。作为一名即将离校的大三学生, 下文我将对已经就读三年的太原理工大学以及软件工程专业情况进行扫盲

观前提醒🚨:

软件工程专业每年分数都在涨，同时招生也一直在扩招，所以分数基本都在往年的位次以上，请不要到处问多少多少分能不能上, 没有人能给你打包票，除非是骗子，你要做的和能做的只有是认真去填志愿，即便滑了也有其他更好的选择做保障(此处推荐一个报考软件"夸克高考")。

接下来是我整理的高考生最感兴趣的问题，我将其分为以下几个模块

  • 专业概况

  • 生活

  • 学习

  • 就业

  • 毕业去向

🏛️专业概况

官网介绍

软件学院-太原理工大学本科招生网 (tyut.edu.cn)

学费 16000元/年 住宿费 男 1000元/年 女 1200/元年（宿舍情况见生活一栏）

学制是 3年学校基础知识教学 + 1年基地就业指导教学 (就业情况见就业一栏)

软件工程 不可转专业！！！ 由于学生基数大，保研率只有百分之2左右

🏠生活

目前， 软件工程大学三年位于太原理工明向校区

明向校区除了体育馆（在建），游泳馆（在建）， 其他设备基本齐全

教学楼，图书馆等公共场所校园网全覆盖，宿舍无

住宿方面

均为上床下桌，有暖气，阳台，无独立卫浴，无空调，晚11点-早6点宿舍楼准时关电闸（夏天可能延迟到晚上12点）

真实感受是没有空调会有一个月左右的时间的晚上热的睡不着, 偏偏恰逢考试周就有点难受，最好自己配备一个充电宝和电风扇

男生6人间，女生4人间（部分6人间）

男生宿舍楼高6层无电梯， 女生宿舍楼高层有电梯

饮食方面

学校有三个大餐厅，基本三四层，大型超市两个（位于东餐厅和西餐厅下方），小超市一个， 基本保证10分钟以内能从宿舍楼到食堂

娱乐方面

近几年周边商城慢慢发展起来来，各种聚餐，包括娱乐设施都很健全(具体可在美团，抖音搜索“晋中榆次大学城”)🎉

交通方面

学校从东区到西区步行大概需要十五分钟左右，校内有小白龙，校外有共享电动车，据说之后要在学校内部也会有共享电动车🛴

📖学习

太原理工大学大一会有晚自习，从晚上7点到9点

上大学之后，你一定会对B站大学有所耳闻，我大学期间无论是学校课程还是专业技能都是在B站学的，各种优秀的学习视频资源几乎都可以在B站找到。

专业课程包括课时的安排由学院制定，2021级往上是一套教学计划，2022级以及往下教学计划是另一套（课程变化不多，基本方向是缩短课程时长，给学生更多自由支配学习的时间），软件工程本科教学计划和计算机科学与技术非常接近，大一基本都是通识课（数学，英语，物理）+部分专业课，大二，大三重点学专业课。

下面我将分为通识课和专业课进行讲解

通识课

数学是工科的基础，作为工科的一员，太理软件工程需要学习的数学课程有如下

  • 高等数学
  • 线性代数
  • 概率论和数理统计
  • 离散数学

尤其考研的同学，前三门最好认真学一遍（虽然大三准备考研的时候也会忘记不少，但有基础复习起来更高效一些）

英语的话，只有大一开设上下两个学期，基本难度也就是四级的水平，考前好好复习就没问题

大学物理只在大一下开设，大二会有实验课，这门比较难，挂科率也比较高，要好好学

专业课

无论考研还是工作，计算机专业课程四（五）大件是一定要重点掌握的，排名顺序与优先级相关

  • 数据结构与算法
  • 计算机操作系统
  • 计算机网络
  • 计算机组成原理
  • 编译原理

软件和计科相比，没那么偏重硬件，所以课程学习上 计算机组成原理 替换成了更为简单的 计算机硬件基础

如果你是就业的话，特别是开发方向的，这门课也是很重要的

  • 设计模式

额外学习任务

目标就业或者深耕本专业想打竞赛的同学课下可以额外去学习算法至少了解一门计算机语言的语法和集合工具 + 数据结构的基础），无论找工作，考研机试都绕不开这个话题

刚刚高考完的同学可以好好去享受这个暑假，但有余力的同学想要提前学习专业知识的同学，我推荐两个学习方向

  • C++基础 + 数据结构

或者

  • html+ css + javascript

第一个是让刚步入大学的同学打好深入学习算法的基础，但难度比较大

第二个是给刚步入大学的同学提高对计算机的兴趣，更加偏向应用方面，稍微轻松一些

暑假一共就2-3个月的时间，出过闲杂繁琐的各项事宜，刚开始学习一门编程语言，就可能就要花费掉你一个多月的时间，这样的暑假不可多得，还是希望大家去尽性的去玩，学习的时间大学有的是，只是想给因为自己高考失利焦虑的同学提供一个正向释放的途径

奖学金方面

国家奖学金： 8000元/年

国家励志奖学金： 5000元/年

优秀学生奖学金：

  • 一等 900元/学期
  • 二等 600元/学期
  • 三等 300元/学期

学科竞赛奖学金，根据每年学科竞赛等级和学校政策动态调整

💼就业

软件工程专业大三上12月份左右会选择相应的方向，目前一共有五个方向

  • 软件开发
  • 移动互联
  • 信息安全
  • 大数据
  • 人工智能

大四6-7月份会对实训企业进行投标，7月中上旬会确定企业和实训地点，7月下旬会前往基地进行实训一直到12月份

关于实训更加具体的信息， 我准备之后单独再出一篇来帮助今年大三的同学抹平信息差，选择自己合适的方向

其次可以关注 太原理工大学软件学院公众号, 这个公众号每年6~7月份会发布一个叫"实训成长明星"的栏目, 直接进入公众号搜索"明星"两个字即可, 重点关注"就业明星" 和 “考研明星” 这两类文章, 基本展示的就是太理软件毕业生中佼佼者的去向

由此可以了解到太理软件就业的上下限区间其实非常大

软件学院由于招生规模较大，学生综合素质相差较大，当然学生自身在校期间的积极性相关，但这些因素综合影响下的结果就是，实际就业水平天差地别，有的同学大二大三就积极参加各类专业竞赛，准备相关技能以及面试技巧，投身到企业中实习，无论实习还是技术广度和深度都碾压同届，自然大厂offer拿到手软，赢得者通吃，有的同学一连挂科几门，学业预警，无实习经历，无技术，无荣誉奖项，无亮眼的校园经历，大四去了基地面临秋招也依旧没有紧迫感，甚至连工作也找不到，但凡去基地后努努力，就算拿不到开发岗offer，拿个交付或者外包offer也不成问题

🎓毕业去向

软件专业因为就业不错，所以我单独开了一栏讲，除过本科毕业直接就业的同学，其实和其他专业差不多，基本都是分流这几个去向

  • 考研
  • 考公
  • 考编
  • 出国

目前互联网行业自2021年最后的辉煌过后，2022年以及往后就业市场愈加不景气，并且互联网行业虽然有着高薪的标签，但工作的996，压力大也是常态，虽说 211的title + 应届生找一份糊口的工作并不是难事，但明显感觉到身边有考研意向的同学是在增加的，通过提升自身学历缓冲就业压力也是不错的选择
